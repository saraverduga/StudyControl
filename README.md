# StudyControl

## Descripción
StudyControl es una aplicación desarrollada en Python para ayudar a los estudiantes a organizar sus tareas académicas y fechas de entrega.

## Objetivo
Facilitar el control y organización de tareas estudiantiles mediante una aplicación sencilla y fácil de utilizar.

## Problema que resuelve
Muchos estudiantes olvidan tareas o actividades importantes debido a la falta de organización académica.

## Público objetivo
Estudiantes de colegio y universidad.

## Funciones principales
- Registrar tareas
- Visualizar tareas
- Organizar actividades académicas

## Tecnologías utilizadas
- Python
- Visual Studio Code
- GitHub
